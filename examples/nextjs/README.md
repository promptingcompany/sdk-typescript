# NextJS Sample

## Setting up local development

1. Link the root sdk with `cd ../../ && bun link`
2. Change directory back to this folder then bun install `cd examples/nextjs && bun i`

## Getting Started


First, run the development server:

```bash
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
